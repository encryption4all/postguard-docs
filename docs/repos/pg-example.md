# pg-example

[GitHub](https://github.com/encryption4all/pg-example) · TypeScript · Example Application

Standalone example application demonstrating PostGuard encryption for strings and files using the WASM bindings directly. Uses Webpack for bundling.

For higher-level integration examples using the `@e4a/pg-js` SDK, see [postguard-examples](/repos/postguard-examples).

## Examples

- **String encryption**: `examples/string.js`
- **File encryption**: `examples/file.js`

## Running

```bash
yarn install
yarn dev     # start webpack dev server
yarn build   # production build
```
